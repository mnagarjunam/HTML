About The Project

Home Automation using IOT and Arduino. The main objective of this project is to build a smart home device which can be used to control the home appliances via internet.This establishes the internet connection to the system and all the home appliances can in turn be connected and controlled by internet.To facilitate the wireless connectivity with the system, the Arduino Uno will be embedded with a WiFi module.
